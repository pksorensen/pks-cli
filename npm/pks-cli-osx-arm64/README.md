# @pks-cli/pks-osx-arm64

This package contains the macOS ARM64 (Apple Silicon) binary for PKS CLI.

## Usage

This package is automatically installed as an optional dependency when you install `@pks-cli/pks` on a macOS ARM64 system.

You should not need to install this package directly. Instead, install the main package:

```bash
npm install -g @pks-cli/pks
```

## Platform Requirements

- **OS**: macOS
- **Architecture**: ARM64 (Apple Silicon / M1/M2/M3)

## Main Package

For documentation and usage instructions, see [@pks-cli/pks](https://www.npmjs.com/package/@pks-cli/pks).
