# @pks-cli/pks-osx-x64

This package contains the macOS x64 (Intel) binary for PKS CLI.

## Usage

This package is automatically installed as an optional dependency when you install `@pks-cli/pks` on a macOS x64 system.

You should not need to install this package directly. Instead, install the main package:

```bash
npm install -g @pks-cli/pks
```

## Platform Requirements

- **OS**: macOS
- **Architecture**: x64 (Intel)

## Main Package

For documentation and usage instructions, see [@pks-cli/pks](https://www.npmjs.com/package/@pks-cli/pks).
